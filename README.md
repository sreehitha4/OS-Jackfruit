OS Jackfruit - Container Runtime Project
Overview 
This project implements a lightweight container runtime in C using Linux system calla.It demonstrates process isolation and file system isolation using chroot.
##Features 
container execution using chroot
process isolation
basic container runtime
##Technologies
c
Linux
GCC
Ubuntu
##How to Run
1.Clone Repo
2.Run make
3.Run container using:
sudo ./engine run c1 ../rootfs-alpha /bin/sh
